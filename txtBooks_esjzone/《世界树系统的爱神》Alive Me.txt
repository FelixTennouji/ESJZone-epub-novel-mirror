简介
20XX年，一款世纪MMORPG游戏破镜而出，然后...我还没想好
前言:世界树的诞生X由来
<System World>系统世界，乃是人类至今算得上是数一数二的世纪大发明。能使人类的意识脱离本体，畅游在万象大千的网路世界。
<System World>创造了许多人梦寐以求的世界，虚拟潜行装置可说是几乎办到了人类所能做的限制。
林林种种的游戏类型出世，让人们生活不再那么的枯燥乏味。甚至可以说是拯救了某些人。
而如今，自<System World>出现至今已经过了五年，<System World>不再像当初那样的杂乱无章，超过百万种的游戏经过了五年的时间，早已没了当初的人气，只因没有那些精序的代码工程，导致了许多游戏时的不顺畅。甚至跌落神坛的游戏不在少数。然而有一款游戏，它奠定了<System World>中第一游戏的地位，至今无人可撼动。
<Tree System >是<System World>出世后一年才发布的MMORPG游戏，可说来也奇怪。在出发点就已经落后的这款游戏，就连发布后也没有大肆宣传，只是静静地等待著人们前来游玩。明明是如此的低调，却成了现在无可撼动的一座大山。
然而直到现在，所有有关制作商的消息却是一条都没有，没错，任何一条。简直就像自行管理的系统一般，让人如此的匪夷所思。
尽管如此，还是拥有许多的人们游玩，而这是游玩过的人们所给出的感想:无法想像它只是游戏。
<Tree System >有著魔法，特性，技能等所有能想到的RPG游戏都有的系统，然而有一点正是<Tree System >占据第一的原因。
就如同它的名字是树系统一样，这款游戏的剧情就像是树一样，拥有无限的分支就拥有无限的剧情，四年时间还是没有人能完全破关游戏，甚至是让人们感觉到一种「无论如何都无法通关」感觉，这剧情是如此之多，如此之大，就犹如世界一般。
因此<Tree system >也有另一个外号，<World Tree System >世界树系统，无止境的剧情分支延伸至世界，世界的无穷远不是人类所能触及的，才因此被称作<World Tree System >
而且明明游戏是如此之大，所有的细节却能做到完美，明明是NPC却能像真人一般的说话活动，简直完全看不出缺点。
尽管< System World>是创世纪大发明，然而这款游戏的制作却远比这项发明要来的恐怖，无人能得知这款游戏是输入甚么程式代码，是如何让它变成如此的真实，甚至连如何制作都不知道，数年来许多人骇入系统来盗取资料，然而却仿佛有股无形的墙壁阻挡一样，时至今日没有任何一人能探知其中的秘密，因此这也被当作是一个都市传说。
这就是<World Tree System >
无可触及的世界树。
01.为虚为幻的欲望
『欲望』是人们心底最原始的情感。
人都拥有想要的事物，为达目的不惜一切。
欲望为母，欲望就是源源不断动力来源，支撑著需要欲望来达成的一切目的。
然而，接触不到的欲望，就是虚幻的。为虚为幻，亦为不存在。
人类会为了不存在的事务而感到憧憬，会不惜一切代价得到想要的虚幻。而欲望则就是为了此刻存在的，因此虚幻的事物也被称为『都市传说』
—
「你给我出去，我没有你这个儿子。」
「别这样...老公，听孩子解释阿...」
「从今以后我没有你这个儿子。」
玻璃的碎裂声响起，父亲暴怒的将我的物品全部都摔了出去，对于平时只要听母亲说几句气就会消停的父亲来说，想必我很让他失望吧。
我从小便很憧憬女性，对于男生的东西我都不感兴趣。洋娃娃、裙子、化妆品，到最后甚至连『想要变成女生』的念头，渐渐地...变成了愿望。
灵魂活在错误的身体里让我非常的痛苦...甚至让我拥有想要自杀的念头。
...好痛苦，真的好痛苦。
我很讨厌我这副身体，然而我却无法做出任何表态，只因父亲是个很古板的人。
我从未像现在一样感到自己是多么的弱小，甚至连最简单的表态都做不出。
然而那年...
<System World>
出世后，我第一次感觉自己是离愿望是多么的近，甚至触手可及。
我以女性身姿、容貌、声音，开始游玩潜行游戏，我的心情从未达到过如此的兴奋。
男生的汗味变成了女生独有的体香、不再拥有粗犷的身体，取而代之的是吹弹可破的肌肤与女生娇小的身躯。
不再拥有冲破天际的棒状物而是变成了令人垂涎的洗面乳和吸引万千男生的粉红鲍厢。
阿...真香~
『现在开始深入教程。』
一道让人身子酥麻的抚媚声音传来，想必那就是这个游戏的引导NPC了。
然后我的身体(虚拟)便开始了自我运动…
「嗯…阿…呼呼。」
淫荡的娇喘声回荡在空气中。
我的手不受控的开始抚摸自己，一只手揉著胸器、玩弄著自己的乳头，身子仿佛像是触电了一样，完全使不上任何力气。
「…这就是…女性的身体…吗?」
好舒服，这完全不是男性的身体可比拟的。
此时，另一只手却开始慢慢地往下抚摸，慢慢的…慢慢地。
「阿。」
在手指触碰到荳荳的那刻，一股从身体深处犹然而生的兴奋感涌起，此刻脑子完全空白，整个身心都沉溺在这份快感之中，无法自拔。
此刻手指不自主地开始加快力道与速度，如果说先前的速度只是前菜的话，现在就是开始品尝主菜的部分。
手指自动伸入了蜜穴的深处，仿佛堪比金手指加藤桑的手部技巧，淫荡的水声在蜜穴里滋滋作响，在兴奋感达到极致的那个瞬间，仿佛无法感受到任何事物一般，只沉溺在那无尽的轮回之中…
而五年后的今天，也就是我下定决心向父亲说明变性的事情。
然而父亲却坚决不同意，我从未看过父亲如此的生气，在他把所有的玻璃制品摔出的时候，我的心…便随著它一起破碎了…
徬徨无措的我走在街道上，像是映衬我的心情一样，此刻大雨滂沱，顺著脸颊所滴下的眼泪甚至在一瞬间就和雨滴融合了。
身处在绝望谷底的我看像了谷顶，像是要寻找那道微小希望一般，就像死命抓著救命稻索的人一样，我抬头看著天空，意识…却在此刻消逝无踪。
02.陌生之人
「处理好了吗？」
空洞、虚无、缥缈，从黑暗的深处传来的声音，那是令人打从心底感到冷颤的声音，一不注意就仿佛会被吞噬殆尽一样。
恐怖不已，光是声音就能令人感到如此，能散发出这种感觉...绝对不是人类。
「嗯，毕竟是欠你的人情，没事的话我就先走了。喔，对了，我很期待那天的到来。」
「没事的话就走吧，无论发生什么事，在她回来之前，我都不会出手。」
如果说深处传来的声音是虚无缥缈的，那么此刻令一位则是给人刚勇坚毅的感觉，就像傲然矗立的山峰一样，永垂不朽。
从两人之间的谈话中不难看出，他们口中的她，应该是对两人来说很重要的存在吧。
与此同时，某处消失的意识再次醒来…
—
「呐。」
究竟活著还有什么意义呢？
如果说错误的灵魂活在了错误的身体里，强行驱使著不合零件的器械，终有一日还是会毁坏。人亦是如此，虽说我生为男生，但追求女性又有何不对。
一昧的阻挡人的希望，只会让人活下去的动力越来越小。
「呐。」
为何父亲不同意我的想法，真正让我完成心愿又如何。他会损失什么吗？钱？面子？这有比他们唯一一个孩子内心的想法珍贵吗？
虽说没有父母亲就没有我，但…真的破坏了一个人活下去的希望，那么…就如同行尸走肉般…令人唏嘘。不如死了才是解脱。
「呐！我说！！！」
咦？
因为突如奇来的声音而打断了我的思考，话说，我连有没有人在旁边都不知道。
「妳终于注意到我了，在想什么呢？想的那么入迷。」
「没有，没什么。」
几乎是下意识的说出，虽说不知为何要否认就是了。
「那个，妳是谁？为什么在这里？」
好美丽…该说是气质吗？看了她仿佛就会被吸引一样，有种无形的吸引力在牵扯著我，好似一不注意便会爱上她一样。
「啊，讨厌，可别爱上我阿，不过我可不讨厌自己爱上自己。」
自己爱上自己，什么意思？
话说，她为什么会知道我在想什么？
然而就在我的大脑还没来得及运转时。不如说，我连自己在哪都不知道，连她是谁都不知道，我的意识又再次消散了。
「不过…是哪里出错了吗？我的最后一个人格…算了，总会恢复的，或许…这次会有不一样的结果呢。」
留下这么一句令人不解的话，人就消失了，声音飘散在空气中，渐行渐远。
